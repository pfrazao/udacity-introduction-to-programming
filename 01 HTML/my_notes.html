My HTML notes

The HTML stands for HyperText Markup Language.

The Hyper on HTML tells how the browser should display the text.

The nesting concept looks like this: E=mc2.

Regarding the lists there can be an ordered list:

  1. Element 1
  2. Element 2
  3. Element 3
Or an unordered list:
  • Element 1
  • Element 2
  • Element 3

For the nested list one can have:

  • Mammals
    1. Raccoons
    2. Gorillas
  • Reptiles
    1. Iguanas
    2. Cobras
  • Birds
    1. Ostriches
    2. Ravens

In HTML the HT stands for hypertext since the pages can link to between them. For instance, a link can be one such as Google.
